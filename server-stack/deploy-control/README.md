# server-stack/deploy-control

Deployment orchestration crate (`deploy-control`).

## Responsibilities

- Implements release application workflow and validations.
- Integrates deploy operations with filesystem, DB, and nginx layer.
- Provides reusable control-plane logic used by server APIs.
