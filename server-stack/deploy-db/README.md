# server-stack/deploy-db

Persistence crate for deployment metadata and migrations.

## Responsibilities

- Defines schema access for deploy/control state.
- Hosts migration logic used by server-side services.
- Provides DB access utilities for PostgreSQL/SQLite scenarios.
