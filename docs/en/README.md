# PirateServer Documentation (EN)

## Quick Start

- Local client stack: [`docs/en/local-client/README.md`](local-client/README.md)
- Server client stack: [`docs/en/server-client/README.md`](server-client/README.md)

## Module Map

- `local-stack` - operator workstation tools: CLI, desktop UI, local agent runtime.
- `server-stack` - server-side services: gRPC deploy, HTTP control API, web dashboard, install scripts.

## Reading Order

1. Start with the target stack (`local-client` or `server-client`).
2. Open the `README.md` inside each module directory.
3. Follow cross-links to related modules for the full deploy flow.
