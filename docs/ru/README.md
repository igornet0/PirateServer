# Документация PirateServer (RU)

## Быстрый старт

- Локальный клиентский стек: [`docs/ru/local-client/README.md`](local-client/README.md)
- Серверный клиентский стек: [`docs/ru/server-client/README.md`](server-client/README.md)

## Карта модулей

- `local-stack` - инструменты оператора на ПК: CLI, desktop UI, локальная агентная логика.
- `server-stack` - серверные сервисы: gRPC deploy, HTTP control API, веб-панель, скрипты установки.

## Как читать документацию

1. Начните с нужного стека (`local-client` или `server-client`).
2. Откройте `README.md` нужного модуля в самой папке.
3. Переходите по ссылкам на соседние модули для полного сценария deploy.
