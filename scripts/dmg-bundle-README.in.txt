Install
-------
1. Drag the application into your Applications folder.
2. If macOS shows a security warning on first launch, right-click the app, choose Open, then confirm. You can also allow it under System Settings → Privacy & Security.

Command-line tool (pirate), if provided with your build, may need to be added to PATH — see the project documentation.
