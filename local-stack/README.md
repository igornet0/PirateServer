# local-stack

Local workstation side of PirateServer.

## Modules

- [`client`](client/README.md) - CLI for pairing, upload, deploy, and operational commands.
- [`desktop-client`](desktop-client/README.md) - Rust backend commands used by Tauri desktop app.
- [`desktop-ui`](desktop-ui/README.md) - desktop UI frontend and Tauri shell.
- [`local-agent`](local-agent/README.md) - local automation agent module.
