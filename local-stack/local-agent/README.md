# local-stack/local-agent

Local agent runtime module for workstation-side automation scenarios.

## Responsibilities

- Hosts long-running local workflows (planned/experimental).
- Coordinates local environment tasks that do not belong in CLI.
- Provides an extension point for gateway/tunnel style integrations.
