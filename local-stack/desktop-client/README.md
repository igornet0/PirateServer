# local-stack/desktop-client

Rust shared backend crate (`pirate-desktop`) for desktop application commands.

## Responsibilities

- Implements Tauri command-side business logic.
- Stores connection state and local desktop metadata.
- Bridges UI actions to gRPC/control API operations.
